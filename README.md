## Redesigning the website for Military Connected Students

### Goals:

- #### Improve Readability and Accessibility

  - Users should not have to make contact if their question is easily answered

- #### Adding a chatbot for an efficient streamlined FAQ

  - Chatbot will contain knowledge relevant to specific student requirements and benefits
  - Will guide students to more information if needed

- #### Improve visual appeal and clutterness

  - Information overload can cause users to resort to calling and emailing which slows down form processing by staff
